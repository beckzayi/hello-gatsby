# Get a Pet

<hr />

When reading a Pet, you must specify ```petId``` property.

1. Click "GET" button.
2. Click "Try it out".
3. Enter ```petId```, for example 2.
4. Click "Execute". And you will see "Loading" while waiting.
5. Response will display after a few seconds if it is the first time to load.
